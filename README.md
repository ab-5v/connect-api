# connect-api

 Connect-api is a simple REST JSON middleware for [connect](http://www.senchalabs.org/connect/).

 See `test/fixtures/index.js` for example of usage.
 
 Installation:
 
    $ npm install connect-api 
 
 Running tests:

    $ npm install
    $ node test/index.js
    $ make test
